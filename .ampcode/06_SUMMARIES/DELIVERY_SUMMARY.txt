EXECUTION FLOW:
═════════════════════════════════════════════════════════════════════════════

BEFORE (Old):
  runTool() → POST /v1/tools/{id}/run [SYNC] → immediate result

AFTER (New):
  apiClient.analyzeImage() → POST /v1/analyze → returns job_id
  apiClient.pollJob() → GET /v1/jobs/{id} → polls until done
  State Machine: queued → running → done → error
  UI shows status at each transition
